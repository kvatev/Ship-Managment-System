# Ship Management System


## 📜 Описание

**Ship Management System** е уеб апликация, предназначена да подпомага управлението и организацията на кораби. Проектът е разработен в рамките на обучението ми във [ВВМУ "Никола Йонков Вапцаров" - Варна](https://www.naval-acad.bg/) . 

## ✨ Основни функции

- Управление на данни за кораби
- Следене на екипажи и маршрути
- Планиране и управление на поддръжка
- Генериране на отчети и анализи

## 🛠 Технологии

- **Език:** C#
- **Фреймуърк:** ASP.NET Core
- **База данни:** SQL Server
- **Фронтенд:** HTML, CSS, JavaScript
- **Интеграции:** API за корабни данни (при нужда)

## 📦 Инструкции за Инсталиране на Ship Management System

1. **Клониране на репозитория:**
   ```bash
   git clone https://github.com/kvatev/ship-management-system.git
   
2. **Уверете се, че имате инсталиран .NET SDK.**
   ```bash
   winget install Microsoft.DotNet.SDK.8
   
3. **Стартиране на приложението:**
   ```bash
    dotnet run
5. **Достъп до уеб приложението:**
Отворете уеб браузър и въведете http://localhost:5000, за да достъпите приложението. Можете да се регистрирате като нов потребител или да използвате съществуващ акаунт, за да влезете.
